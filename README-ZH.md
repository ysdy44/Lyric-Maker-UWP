![](ScreenShot/logo.png)

 
 Windows 10 应用商店地址:

## 功能说明

歌词文件编辑工具，可以编辑和移动歌词，支持预览、播放、速度、可视化和时间线。 如果你需要编辑歌词不妨试试。


<br/>

## 开发环境

|Key|Value|
|:-|:-|
|系统要求| Windows10 Creators Update 或更高|
|开发工具|Visual Studio 2019|
|编程语言|C#|
|显示语言|中文 and English|
|评论语言|中文|


<br/>

## 部署说明

> 下载整个工程，运行`Lyric Maker.sln`，启动工程`Lyric Maker`，试试看是否可以工作。
